## Starting up the Web Server

> rails server 			-> linux
> ruby bin\rails server -> windows
> rails s 				-> short 

> rails generate controller Welcome index
> the controller located *app/controllers/welcome_controller.rb*
> the view *app/view/welcome/index.html.erb*
